시장 대시보드에 **TQQQ**를 추가하고, 고점 대비 하락률에 따라 **텔레그램으로 매수 타이밍 알람**을 보내는 기능을 새로 붙였습니다. 코드는 `master`에 반영 완료.

## 이번에 한 일 (완료)

- **TQQQ 추가**: 웹 화면에 TQQQ 카드(현재가·등락률·고점대비 하락률) 추가. 한투 해외주식 API로 시세를 가져오고, 안 되면 야후 파이낸스로 대신 가져옵니다.
- **투자 타이밍 알람**: `trigger_engine.py` 새로 만듦.
  - 코스피·코스닥: -30/-40/-50/-60/-70% 단계별 알람 (S&P500 동반 하락 시 미국 배분 안내)
  - TQQQ: 1차(-10%~-49%, 40회), 2차(-50% 아래 가격 10등분)
  - 신고점 리셋, -50% 재진입 시 기준가 재설정, 중복 알람 방지까지 포함
- **알람시계 확장**: `scheduler.py` 가 평일 15:30 브리핑에 더해, **10분마다 알람 조건을 검사**합니다.
- **웹 화면**: "🔔 투자 타이밍 현황" 섹션 추가 (`/api/alerts` 주소).
- **백테스트**: `backtest.py` 로 과거 데이터에서 알람이 언제 울렸을지 연습 가능.
- **문서**: `README.md`, `CLAUDE.md`, 이 파일 정리.

테스트: 알람 로직은 가짜 데이터로 다 통과했고, 웹 화면도 실제로 사진 찍어 확인했습니다.

## 사용자가 직접 해야 하는 일 (아직 안 됨)

봇이 아직 **텔레그램과 연결되지 않았습니다.** 아래 3가지만 하면 됩니다.

1. 텔레그램에서 **@BotFather** 에게 `/newbot` → 봇 만들고 **토큰** 받기
2. 만든 봇에게 아무 말이나 한 번 보내고, 브라우저에서
   `https://api.telegram.org/bot<토큰>/getUpdates` 열어서 `"chat":{"id": ...}` 숫자(= **방 번호**) 확인
3. `.env` 파일에 `TELEGRAM_BOT_TOKEN` 과 `TELEGRAM_CHAT_ID` 넣기
4. `python scheduler.py --test` 로 톡 오는지 확인 → 되면 `python scheduler.py` 로 상시 실행

## 다음에 하면 좋은 일 (후보)

### 1. "지금 당장 보여줘" 기능 (쉽게 말하면)

지금 봇은 정해진 시간에만 톡을 보냅니다. 여기에 **명령 한 번 하면 즉시 반응**하는 기능을 붙일 수 있습니다.

- "지금 시장 캡처해서 보내줘" → 기다리지 않고 바로 사진 전송
- "지금 알람 상황 알려줘" → 코스피/TQQQ가 몇 단계까지 왔는지 글자로 요약

이게 있으면 오픈클로 같은 **대화형 봇**과 붙였을 때 "물어보면 답하는 비서"가 됩니다.
(구현 방법: `app.py`에 즉시 실행용 주소를 추가하거나, `scheduler.py`에 명령어 옵션을 추가)

### 2. 오픈클로 연동 (선택)

- 정확한 숫자 알람(-30%, P 10등분 등)은 지금 코드가 계속 담당 → 정확·저렴·안정
- 오픈클로는 "지금 상황 요약", "왜 빠졌는지 뉴스랑 같이" 같은 **말로 물어보는 것** 담당
- ⚠️ 주의: 오픈클로와 이 프로그램이 **같은 봇 토큰으로 동시에 메시지를 받으면 충돌**합니다. 봇을 2개로 나누세요.
